# Dropdown Menu

Menu anchored to button's bottom edge. Aligns perfectly on left side using CSS anchor positioning.

## Features

- Anchors to button's bottom edge
- Smooth slide-down animation
- Pure CSS implementation
- Opens on focus/active state

## Usage

### Direct Integration

Copy the styles and markup directly from `index.html` into your webpage for full control over the implementation.

### Customization

All customizable values are defined as CSS variables in the `:root` selector:

- `--dropdown-bg` - Dropdown background color (default: #1a1a2e)
- `--dropdown-border` - Dropdown border color (default: var(--brand-blue))
- `--dropdown-border-width` - Dropdown border width (default: 2px)
- `--dropdown-border-radius` - Dropdown border radius (default: 8px)
- `--dropdown-gap` - Gap between button and dropdown (default: 8px)
- `--dropdown-min-width` - Minimum dropdown width (default: 180px)
- `--dropdown-transition` - Transition timing (default: all 0.3s)
- `--menu-item-padding` - Menu item padding (default: 12px 16px)
- `--menu-item-hover-bg` - Menu item hover background (default: rgba(18, 150, 230, 0.2))
- `--button-bg` - Button background color (default: var(--brand-blue))
- `--button-text` - Button text color (default: var(--brand-black))
- `--button-padding` - Button padding (default: 12px 24px)
- `--button-border-radius` - Button border radius (default: 8px)
- `--button-font-weight` - Button font weight (default: 600)
